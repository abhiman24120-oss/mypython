# MyPython - Playwright Sample

This repository contains Playwright sample scripts for browser automation testing in Python.

## Installation

1. Install Playwright:
```bash
pip install -r requirements.txt
```

2. Install browser binaries:
```bash
playwright install
```

## Usage

Run the sample script:
```bash
python playwright_sample.py
```

## Examples Included

- **Basic Navigation**: Navigate to a website and capture page title
- **User Interactions**: Click buttons, fill forms, and press keys
- **Assertions**: Test page content with assertions
- **Multiple Pages**: Handle multiple browser pages/tabs
- **Error Handling**: Implement robust error handling
